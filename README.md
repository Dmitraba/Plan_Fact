# План/Факт по всем проектам

Файлы:
- MASTER_PROJECTS_TIMELINE.csv — единая таблица задач/стадий/дедлайнов по всем проектам.
- PROJECTS_PLAN_FACT_DASHBOARD_VIBE.html — локальный визуальный дашборд план/факт (квартальный шаг).

Как использовать:
1) Открой HTML в браузере.
2) Если обновил CSV — загрузи его через кнопку выбора файла.
3) Фильтруй по отдельному проекту или смотри сводку по всем.

Библиотеки эффектов:
- React Bits: https://reactbits.dev/
- 21st.dev: https://21st.dev/home
