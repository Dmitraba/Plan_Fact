## Fixed UI Rules (No Deviations)
1. One clear hierarchy: title -> quarterly KPIs -> controls -> project cards -> details.
2. No random accent colors: primary blue, neutral slate, amber for in_progress, red only for risk.
3. KPI text must be high-contrast and large enough for quick scan.
4. Cards must communicate status in under 2 seconds.
5. Timeline/quarter block must always show plan %, fact %, delta.
6. Mobile readability is mandatory (single-column fallback).
7. No decorative effects that reduce readability.
